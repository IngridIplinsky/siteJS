Trabalho desenvolvido pela aluna Ingrid Iplinsky de Oliveira

https://ingridiplinsky.github.io/webSite/

O trabalho Web contém 5 paginas:
1- Index - pagina de apresentação da empresa, contendo os valores da empresa, os serviços prestados e contato

2- Quem Somos - Contém um pouco sobre a empresa.

3- Serviços - Contém a descrição dos serviços prestados.

4- Produtos - Contém o portfólio de alguns produtos que a empresa trabalha.

5- Contato -  Contém dados de contato da empresa e formulário para preenchimento.

O projeto contém 2 pastas, 1 pasta para as imagens e outra pasta style com 6 arquivos .css.
